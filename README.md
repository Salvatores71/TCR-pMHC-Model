# TCRpMHCmodel

## Project Setup

```sh
node.js v22.3.0
python v3.9.6
```

```sh
npm install
pip install sys
pip install SequenceQuery
pip install requests
pip install time
pip install os

```

### Project Start

```sh
前端在src中，后端在server中
前端启动：
npm run dev 
后端启动：
cd server
node server.js run

后端指定在端口3000运行，若停止运行后要再次运行，先执行killport.sh 关闭端口3000的进程
server/killport.sh
```
